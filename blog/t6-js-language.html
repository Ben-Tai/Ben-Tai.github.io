In the JavaScript language, describe all the use cases
for each of:

Parentheses()

  1. To execute a function var result = doSomething()
  2. To surround conditional statements if (i = 0){
  3. To group and order expressions a = (i % (1 + 2)) * (3 + 4)

Brackets []

  1. To create a property of an objecct myDog["barks"]
  2. To create an array var list = [0,1,2,3];
  3. To access objects inside an array alert(list[2]);

Braces {}

  1. To define start and end of a function doSomething(){ do(this,that) }
  2. To define start and end of a conditonal statement if { this } else { that }
  3. To define intial values in an arrayint[] a = {5,10,15,20}
