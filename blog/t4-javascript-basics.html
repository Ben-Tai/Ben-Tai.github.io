How does JavaScript compare to HTML and CSS?

HTML tells you were things belong on the page, CSS tells you how the look and JAVASCRIPT makes them interactive e.g. website accounts, online purchases, or that search box in the corner.

Explaination of Flow control and loops

So JAVASCRIPT is all about actions and 'Flow control' is about controlling the flow of actions until you reach a certain end point. For example if someone wanted to loose 10kgs they could decide to do this buy going to the gym.

  1. You would first have to weigh yourself so you know when you have reached your end goal of 10kgs;
  2. You would pick a gym you want to go to;
  3. Go to the gym;
  4. Get a gym membership;
  5. Workout at the gym;
  6. Weigh yourself again to see how much you had lost since your last weighin;
.

This is where 'Loops' come in. You could loop steps[3,5,6] until you have meet your end point of -10kgs

Accessing data from Arrays and Objects

To access and modify data in Arrays you access the indexes[position] of that data in the Array
Objects are similar to arrays, except that instead of using indexes you access the data in objects.properties or object[property]
